    Praktische Erfahrungen

    Softwareprojekt mit Baker Hughes

    3D Visualisierung einer Bohrplanung - Ölbohrstrang

    Bei diesem Projekt wurde eine Software entwickelt, die eine Öl- Bohrlochplanung als Web Applikation darstellen konnte. Die dazugehörigen Daten lagen in einem prioritären Dateiformat vor, die aus vorgelagerten Software Suites stammten. Die so entstandenen Bohrlochdaten wurden mittels 3D und 2D Visualisierung oder in tabellarischer Form dargestellt. Ziel der entwickelten Software war es, den Genehmigungsprozess einfacher zu gestalten und so den Projektteilnehmern die Mitzeichnung zu erleichtern. Die Software konnte zu diesem Zweck Kommentare von Projektteilnehmern zu mehreren Bohrlochplanungen speichern und für andere Teilnehmer sichtbar machen. Mit dem Einsatz der entwickelten Software konnte der Genehmigungsprozess deutlich effizienter gestaltet werden.
